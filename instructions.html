RULE BOOK

  • You have to select 11 players.
  • Different combinations of players has to be within the 100 credit cap and must qualify the following team selection criteria:

    • Type min max
    Wicket Keeper 1 4
    Batsmen 3 6
    All Rounder 1 4
    Bowler 3 4

  • Bowling Points

    • Type Points
    Wicket +25
    Bonus (LBW / Bowled) +8
    4 Wicket Bonus +4
    5 Wicket Bonus +8
    Maiden Over +8

  • Batting Points

    • Type Points
    Run +1
    Boundary bonus +1
    Six Bonus +2
    Half-century Bonus +4
    Century bonus +8
    Dismissal for a duck -3

  • Any player scoring a century will only get points for the century. No points will be awarded for their half-century.
  • If any runs are scored on an overthrow, points for those runs will be credited to the batter on strike for that ball. However, if the overthrow goes for a boundary, the batter will not receive extra Boundary Bonus points.
  • Fielding Points

    • Type Points
    Catch +8
    3 Catch Bonous +4
    Stumping +12
    Run Out (Direct hit) +12
    Run Out (Not a Direct hit) +6

  • Some Extra Points

    • Type Points
    Captain 2x
    Vice-captain 1.5x
    In Starting 11 +4

  • Economy Rate Points(*Min 5 Overs To Be Bowled)

    • Type Points
    Below 2.5 runs per over +6
    Between 2.5 - 3.49 runs per over +4
    Between 3.5 - 4.5 runs per over +2
    Between 7 - 8 runs per over -2
    Between 8.01 - 9 runs per over -4
    Above 9 runs per over -6

  • Strike Rate (Except Bowler) Points (*Min 20 Balls To Be Played)

    • Type Points
    Above 140 runs per 100 balls +6
    Between 120.01-140 runs per 100 balls +4
    Between 100-120 runs per 100 balls +2
    Between 40-50 runs per 100 balls -2
    Between 30-39.99 runs per 100 balls -4
    Below 30 runs per 100 balls -6


  • Negative points for low batting Strike Rates are only applicable for individual Strike Rates of 50 runs per 100 balls or below.
